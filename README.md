## インストールと実行手順

以下の手順に従って、このプロジェクトをローカル環境で実行できます。

### 1. リポジトリをクローンする
```bash
git clone https://github.com/23jy0115/xss-ctf.git
```
### 2. プロジェクトディレクトリに移動

### 3. 依存関係をインストールする
```bash
npm install
```
### 4.サーバーを起動する
```bash
node app.js
または
npm start
```
### 5.ブラウザにアクセスする
[http://localhost:3000](http://localhost:3000)
